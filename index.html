Work

Professional

Security Data Workbench (SDW) – Developer & Cloud Integrator

I contributed to the development and optimization of the Security Data Workbench (SDW), a custom-built web application designed to empower the Security Operations Centre (SOC) analysts with intuitive access to security telemetry and incident data. The application features a streamlined UI layer that connects directly to Azure Data Explorer (ADX) in the backend, enabling fast queries and rich visualizations of complex datasets. Leveraging Angular and Node.js, I enhanced front-end usability and backend performance while ensuring secure data pathways. The project strengthened my expertise in KQL, ADX schema design, API integration, access controls, and responsive dashboarding all delivered within a scalable, cloud-native environment built on Azure.

End-to-End Data Pipeline Development

I have led the design and implementation of robust, scalable data pipelines across multiple Azure services to support security analytics and operational reporting. These pipelines ingested, transformed, and enriched high-volume (petabytes) telemetry and log data using Azure Data Factory, Databricks, Logic Apps, Event Hubs, Stream Analytics and Azure Functions. I applied best practices in schema design, error handling, and performance tuning, while integrating with downstream systems such as Azure Data Explorer, Azure SQL Pools and Power BI for real-time visualization. My work ensured reliable, secure, and efficient data flow from source to insight, enabling the SOC teams to act on threats with speed and precision.

Cloud Architecture & Platform Engineering

Beyond traditional data engineering, I frequently took on cloud architecture responsibilities like provisioning infrastructure, designing secure access models, and optimizing resource configurations for cost and performance. I authored and maintained ARM templates and deployment pipelines in Azure DevOps, implemented role-based access control (RBAC), Access Control Lists (ACL) and advised on network topology and integration patterns. This hybrid role allowed me to bridge engineering and architecture, ensuring that our solutions were not only functional but also scalable, secure, and aligned with enterprise standards.
