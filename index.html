Work

Award Winning Atos Sponsored Project

With the guidance of our project supervisor, Peter Blanchfield, we created a water monitoring system for Atos in which we won the school prize for the best sponsored project. This had three main elements which were the desktop app, the mobile app and the simulation of the water system which was made in MatLab. Personally, I helped work on the mobile app and desktop app in regards to the design and functionality of the TCP IP communication between the two applications.

We took the simulation data from MatLab and used it to display a live simulation of the water system in both the desktop and mobile application:

Using TCP/IP Server and Client Sockets the simulation data could be displayed in both the desktop app and mobile device app. Source Code
